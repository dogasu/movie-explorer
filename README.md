Movie Explorer is a modern React app built with Vite that lets users search for movies, see trending films, and view detailed information including trailers.

## Features
- Search movies using TMDB API
- View trending movies
- See movie details, ratings, release date, and overview
- Watch YouTube trailers
- Recent search history
- Popular movie suggestions
- Responsive and visually appealing UI

## Technologies
- React 18 + Vite
- JavaScript (ES6+)
- HTML / CSS
- TMDB API
